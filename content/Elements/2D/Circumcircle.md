# Circumcircle

Creates a circle that lies on three points. Takes three points as the parameters `[[x,y],[x,y],[x,y]]`.

````yaml
```graph
bounds: [-10, 10, 10, -10]
keepAspectRatio: true
elements: [
	{type: circumcircle, def: [[0,2], [2,1], [3,3]]}
]
```
````
